macOS：照着做了还是打不开
  1. 先确认上一步点的是「完成」而不是「移到废纸篓」。万一点成了废纸篓，去废纸篓还原，或重新下载一次。
  2. 在「系统设置 → 隐私与安全性」最下面找「仍要打开」。如果没看到，先再双击一次安装包让它重新弹一下，按钮就会出现。
Windows：被 360 / 电脑管家 / 杀软拦了
  1. 360 / 电脑管家弹窗：选「允许」或「信任」，不要选「阻止」。
  2. 装到一半失败、或者命令一直打不开，多半是杀软在后台悄悄拦了。打开你的杀软，找到「信任区 / 白名单」，把这个安装器加进去再装一次；实在不会弄，就先临时关掉杀软，装完再打开。
