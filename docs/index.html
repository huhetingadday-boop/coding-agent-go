遇到问题

打不开 / 提示有风险？

macOS：提示「无法打开，来自身份不明的开发者」

本项目免费开源，没花钱给苹果交签名「税」（开发者一年 $99），所以首次打开要手动放行一次，并不是软件有问题。新版 macOS（Sequoia）已经去掉了「右键→打开」这个老办法，请走下面这条，所有版本都管用：

  1. 双击安装包，弹出「无法打开…」时点 「完成」（不要点「移到废纸篓」）。
  2. 打开「系统设置 → 隐私与安全性」，往下拉到「安全性」那一段。
  3. 会看到「已阻止使用 Coding Agent Installer」，点旁边的 「仍要打开」，按提示用密码 / Touch ID 确认一次即可。
Windows：SmartScreen 蓝框 / 杀软拦截
  1. SmartScreen 蓝框：点「更多信息」→「仍要运行」。
  2. 360 / 电脑管家弹窗：选「允许」或「信任」，不要选「阻止」。
  3. 杀软拦了 node 或代理进程：把它加入白名单后重试。
