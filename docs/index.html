打不开 / 提示有风险？

macOS：提示「无法打开，来自身份不明的开发者」

安装包没有花钱买苹果签名，所以首次打开要手动放行（只需一次）：

  1. 把 AI Coding Go 从 .dmg 拖进「应用程序」。
  2. 在「应用程序」里 右键点它 → 打开，再在弹窗里点「打开」。
  3. 还不行就去「系统设置 → 隐私与安全性」，找到拦截提示点「仍要打开」。
Windows：SmartScreen 蓝框 / 杀软拦截
  1. SmartScreen 蓝框：点「更多信息」→「仍要运行」。
  2. 360 / 电脑管家弹窗：选「允许」或「信任」，不要选「阻止」。
  3. 杀软拦了 node 或代理进程：把它加入白名单后重试。
想校验文件完整性 (SHA-256)

每个安装包都附一个 .sha256 文件。对比哈希值一致即可放心运行。

会用命令行？一行直接装（不用下载安装包）

macOS / Linux / WSL — 打开终端粘贴回车：

bash <(curl -fsSL https://cdn.jsdelivr.net/gh/huhetingadday-boop/coding-agent-go@latest/install-gui.sh)

Windows — 打开 PowerShell 粘贴回车：

irm https://cdn.jsdelivr.net/gh/huhetingadday-boop/coding-agent-go@latest/install-gui.ps1 | iex
